## 9. Notas Importantes

### Cambios Recientes en Fuentes Gubernamentales (2025)

1. **CISA (EE.UU.)** - Desde mayo 2025, CISA descontinuo los RSS feeds para:
   - Known Exploited Vulnerabilities (KEV) Catalog
   - Cybersecurity Alerts & Advisories
   - Las actualizaciones ahora solo via email (GovDelivery) y redes sociales (X @CISACyber)
   - Feed legacy disponible: `https://www.cisa.gov/news.xml`, `https://www.cisa.gov/blog.xml`, `https://www.cisa.gov/cybersecurity-advisories/all.xml`

2. **ENISA (UE)** - Descontinuo sus RSS feeds con el redisenio del sitio web. Aun no han implementado un mecanismo de suscripcion alternativo.

3. **NVD (NIST)** - Retiro los feeds RSS legacy en marzo 2024. La API REST 2.0 es el metodo recomendado.

### Recomendaciones para el Agregador

1. **Feeds prioritarios para LinkedIn** (contenido general, confiable):
   - The Hacker News, BleepingComputer, SecurityWeek, Dark Reading
   - Krebs on Security, Graham Cluley
   - CISA Blog (aun activo), NCSC UK

2. **Feeds tecnicos para contenido avanzado**:
   - Google Project Zero, PortSwigger Research, SANS ISC
   - Cisco Talos, Unit 42, CrowdStrike Blog

3. **Feeds de vulnerabilidades para alertas**:
   - NVD API 2.0, CVE Feed, CISA KEV JSON

4. **Para contenido en espanol**:
   - Traducir contenido de fuentes prioritarias en ingles
   - ESET WeLiveSecurity (verificar RSS en espanol)
   - INCIBE (Espana)

### Frecuencia de Actualizacion Recomendada

- **Portales mainstream**: Cada 1-2 horas
- **Blogs de investigadores**: Cada 6-12 horas
- **Fuentes gubernamentales**: Cada 6-12 horas
- **Feeds de CVEs**: Cada 15-30 minutos (para feeds high/critical)
- **Newsletters**: Diaria (coincidir con envio de newsletters)
